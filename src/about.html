Welcome to our webpage! This page is a project created by Dikshit, Skand Dev Dixit and Divyansh Goyal for the operating system course. Our webpage aims to help users calculate the average waiting and turnaround time of processes using different CPU scheduling algorithms. We have implemented algorithms such as First Come First Serve (FCFS), Shortest Job First (SJF), Priority Scheduling, and Round Robin (RR). By using our webpage, users can input the burst time and priority of each process and get the results in the form of a table that shows the waiting time and turnaround time for each process, as well as the average waiting time and turnaround time. Our webpage provides an easy-to-use interface for users to test different scheduling algorithms and compare their results. We hope you find our webpage useful and informative!
